\section{REED command line flags summary}\label{flagsSummary}
The following summary was generated automatically by using \texttt{reed~-summarise}, running REED version 3.2. Compiled 15:55:06, Jun  9 2026.
\vskip 5mm\begin{description}\raggedright
\item[\tt -\#]~\\ show comments in the REED file, useful if comments are used as reminders.

\item[\tt -basename <path/file.ext>]~\\ set pathname for generated files (replacing \texttt{.ext} with \texttt{.gv}, \texttt{.html}, \texttt{.pdf}, etc).

\item[\tt -c]~\\ show weakly connected components.

\item[\tt -allcolors]~\\ list all colors that are available for highlighting.

\item[\tt -colors]~\\ list all colors used.

\item[\tt -colors+]~\\ lists all colors like \texttt{-colors} but also gives the meanings of all colors used.

\item[\tt -F]~\\ highlight flags in graph drawing --- unfortunately due to a GraphViz bug, this breaks up any groups.

\item[\tt -f]~\\ show textual descriptions of flag colors in REED drawing.

\item[\tt -flags]~\\ show all flag definitions.

\item[\tt -g]~\\ generate a GraphViz \texttt{.gv} file --- GraphViz is the default option if nothing else chosen --- See \texttt{https://graphviz.org} for details.

\item[\tt -go]~\\ as \texttt{-g} but also open the \texttt{.gv} file.

\item[\tt -h]~\\ generate an interactive \texttt{.html} REED file.

\item[\tt -ho]~\\ as \texttt{-h} but also open the \texttt{.html} file.

\item[\tt -html]~\\ long form of \texttt{-h} flag.

\item[\tt -htmlo]~\\ long form of \texttt{-ho} flag.

\item[\tt -ids]~\\ show full names and IDs in the REED graph.

\item[\tt -insert <text>]~\\ insert this text to process before the next file.

\item[\tt -json]~\\ generate a JSON \texttt{.js} file with all information from the GraphViz diagram.

\item[\tt -keywords]~\\ list all keywords used.

\item[\tt -l]~\\ generate a Latex REED file --- also generates some useful Latex definition files.

\item[\tt -m]~\\ generate a Mathematica \texttt{.nb} file representing the REED graph as a series of expressions.

\item[\tt -n]~\\ show node IDs in graph drawing.

\item[\tt -o]~\\ open generated files automatically.

\item[\tt -pdf]~\\ generate a \texttt{.pdf} file representing the REED graph.

\item[\tt -pick <color>]~\\ restrict generated files to just this color.

\item[\tt -pick <keyword>]~\\ restrict generated files to notes with this keyword --- the keyword can be abbreviated: use \texttt{...} so \texttt{xyz...} matches keywords or phrases starting \texttt{xyz}.

\item[\tt -pick+ <color>]~\\ same as \texttt{-pick} but also explains this color meaning.

\item[\tt -properties]~\\ generate a \texttt{.csv} file of properties for any application.

\item[\tt -raw]~\\ start in raw mode (skipping text until a start tag) --- only use \texttt{-raw} with \texttt{-tags} flag.

\item[\tt -rules]~\\ summarise HTML-Latex rules.

\item[\tt -s]~\\ list nodes and their full names, sorted by node ID.

\item[\tt -ss]~\\ list nodes and their full names, sorted by full names.

\item[\tt -sss]~\\ list nodes and their full names, sorted by both node ID and full names.

\item[\tt -sep]~\\ draw a separator line before processing any files (useful with \texttt{-watch}).

\item[\tt -sig]~\\ show REED file MD5 signatures.

\item[\tt -summarise]~\\ save Latex overview of REED syntax and command line flags to standard output.

\item[\tt -svg]~\\ generate a \texttt{.svg} file --- representing the REED graph.

\item[\tt -syntax]~\\ summarise REED syntax.

\item[\tt -t]~\\ transpose node numbering --- swap row and column node numbers.

\item[\tt -tags <start> <end>]~\\ only process REED information written between these tags --- you can change tags between files --- and also set tags within a REED file by: \texttt{tags <start> <end>}.

\item[\tt -transit]~\\ list transit nodes and nodes with no influence.

\item[\tt -v]~\\ verbose mode.

\item[\tt -version]~\\ state the version number of the REED command.

\item[\tt -w]~\\ what versions are used in these files? --- helpful to know if using the \texttt{v=} flag.

\item[\tt -watch]~\\ run REED when any used file changes (nice with \texttt{-o} flag).

\item[\tt -x]~\\ generate an \texttt{.xml} file --- representing all REED data for import into other applications.

\item[\tt -xml]~\\ long form of \texttt{-x} flag.

\item[\tt --]~\\ treat all further parameters as filenames --- if you want to have no restrictions on filenames as they otherwise cannot be flags.

\end{description}
